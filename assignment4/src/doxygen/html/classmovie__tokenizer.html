Reads two files and returns a vector of movies.

Parameters
META_DATA_FILEThe filename of the file containing the movie meta data.
DESCRIPTION_FILEThe filename of the file containing the movie descriptions.
Returns
All the movies that have descriptions in the the files.

This function reads all the available data in two files: The META_DATA_FILE and the DESCRIPTION_FILE It then associates the data contained in each by a common piece of data between the two: the Movie ID Then a vector of movies with descriptions is created and returned.
